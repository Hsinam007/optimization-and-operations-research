# Graphical Method of Linear Programming

This folder contains the graphical solution of Linear Programming Problems using Python.

## Contents
- Formulation of LPP
- Graphical solution using Python
- Feasible region visualization
- Corner point analysis
- Types of solutions:
  - Unique solution
  - Multiple (alternate) solutions
  - Unbounded solution
  - Infeasible solution

## Tools Used
- Python
- NumPy
- Matplotlib
- Google Colab

## Learning Outcome
This project builds conceptual and computational understanding of Linear Programming using the graphical method.
